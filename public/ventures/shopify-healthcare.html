The Architecture

Six Modules.
One Unified OS.

Like Shopify's app ecosystem, each module stands alone or integrates into the full stack. Healthcare operators can adopt incrementally or go all-in. The platform generates network effects as data flows across modules.

01

AI Billing Engine

Auto-codes claims from clinical notes using ICD-10/CPT inference. Zero-touch claim submission to all payers. 97%+ first-pass acceptance rate target.

Replaces: Epic Billing, Athenahealth billing module
02

Prior Auth AI

Autonomous prior authorization submission and appeals. Payer policy database updated in real-time. Reduces auth cycle from 2 weeks to 4 hours.

Replaces: Cohere Health, manual auth workflows
03

Claims Router

Distributed clearinghouse routing — no single point of failure. Intelligent rail selection across payer networks. Built-in redundancy where Change Healthcare had none.

Replaces: Change Healthcare clearinghouse dependency
04

Patient Finance

Automated patient balance estimation, payment plans, and collections. Integrates with HSA/FSA and BNPL health financing. Reduces bad debt by 40%.

Replaces: Instamed, Cedar, manual collections
05

Analytics OS

Unified clinical + financial + operational analytics. Real-time P&L per service line, payer contract performance, denial trend analysis. The CFO dashboard healthcare never had.

Replaces: Tableau + manual BI stack
06

Credentialing Hub

AI-automated provider credentialing and payer enrollment. Reduces credentialing from 90 days to under 2 weeks. Single source of truth for provider directory data.

Replaces: Symplr, manual enrollment processes
Design Principle

Composable by Default

Every module exposes a clean API. Operators can use one module standalone or combine all six. Third-party apps can plug in via the healthcare OS API — just like Shopify's 10,000-app ecosystem.

Data Layer

Single Source of Truth

All six modules write to a unified FHIR-compliant data lake. When billing data informs clinical analytics and clinical data improves prior auth accuracy, the network effect compounds continuously.

AI Core

Agents, Not Features

Each module is powered by a purpose-built AI agent trained on healthcare-specific data. Unlike bolt-on AI features, these agents are the product — designed to work without human intervention on 90%+ of tasks.
